# Saber-Translator 漫画翻译器 (Astrbot 插件)

本插件是 `Saber-Translator` 在 `Astrbot` 机器人框架下的专用对接扩展。它允许远程服务器上的机器人实例通过 **Tailscale 局域网直连**，调用运行在本地电脑上的 AI 漫画翻译与修复引擎，进行全自动的漫画图片与压缩包翻译。

---

## 📖 使用说明 (LLM / 大模型阅读专区)

如果你是正在调用此工具的大语言模型（LLM Agent），请仔细阅读以下调用契约：

1. **功能名称**：`translate_comic` (工具函数名)
2. **触发意图**：当用户在聊天中发送了图片或包含了图片的 `.zip` 压缩包，并且**流露出需要“翻译”、“翻译漫画”、“汉化”、“擦除/修复漫画”等相关意图**时，你可以直接决策并调用该工具。
3. **参数传递**：
   - **本工具无需任何参数 (Args 为空)**。请勿尝试生成任何名为 `event` 或包含图片数据的参数，否则将导致参数解析崩溃。
   - 插件本身在后台会自动检索用户在当前会话中已经发送的图片、图片链接或 zip 压缩文件（包含历史缓存的容错读取），无需你手动传递二进制或文件路径。
4. **处理流程**：调用后，插件会将资源上传到本地电脑，依次调用本地的 `detect` (气泡检测) -> `ocr` (文字识别) -> `translate` (文字翻译) -> `inpaint` (背景文字擦除) -> `render` (渲染中文) 流程，并自动把最终的汉化成品图或 ZIP 打包包发回给用户。

---

## 🛠️ 安装与配置说明 (用户阅读专区)

### 1. 配置参数说明
启用插件后，在 Astrbot 管理面板配置项中，您只需填写一项必填项：
- `saber_base_url`：填写您本地电脑在 Tailscale 网络中的 IP 地址和端口（例如 `http://100.115.8.9:5000`）。

其余参数（如服务商 `model_provider`、模型名称 `model_name`、`api_key` 等）**推荐全部留空**。
- **参数留空时**：插件在翻译时会自动向本地电脑请求当前的默认大模型配置（自动同步您在本地网页端已经配好的 API 密钥与模型选择），避免在不同平台重复填写的繁琐。
- **填入参数时**：会强制覆盖本地网页里的翻译配置。

### 2. 调用命令
- **指令直接调用**：在发送漫画图片或漫画压缩包后，紧接着发送 `/trans_comic`，机器人即会启动翻译。
- **对话智能触发**：在已配置多模态 Agent 的机器人会话中，发送图片并打字：“*帮我翻译一下这页漫画*”，大模型即可自动触发并返回翻译成品。
